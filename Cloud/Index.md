## 🗺️ Suggested Learning Path

1. Start with [[Cloud Fundamentals]] to understand the big picture
2. Read [[Cloud Networking Basics]] and [[Cloud Security Fundamentals]]
3. Pick a provider: **AWS** or **Azure**
4. Follow: Getting Started → IAM → Compute → Storage → Databases → Networking
5. Then: Deployment → Monitoring → Security → Billing
6. Use [[AWS vs Azure Comparison]] to map knowledge between providers
