## 🗺️ Suggested Learning Path

### If you know nothing about frontend:
1. Start with [[Frontend Fundamentals]]
2. Pick a framework: React (simpler start) or Angular (more opinionated/structured)
3. Follow the notes in order within that framework
4. Read [[React vs Angular Comparison]] to map knowledge between them

### If you want to learn both:
1. [[Frontend Fundamentals]] → React (all 10 notes) → Angular (all 10 notes)
2. Use [[React vs Angular Comparison]] as a bridge between the two
