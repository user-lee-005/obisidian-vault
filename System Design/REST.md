- Most common
- REST - Representational State Transfer
- This is resource based, and uses HTTP methods as the protocols
- This is stateless, contains all the information required to process the request
- This has Standardized Methods, GET | POST | PUT | PATCH | DELETE | OPTIONS | TRACE
- Most common API style
- Used in mobile and web apps
- Explicit versioning
- Built-in HTTP caching
---
Example

`GET /api/v1/users/123`
