When there are 2 tables and need to create a separate table that would connect those 2 tables into a single table which would hold the information about both the tables.

Example:
	- We have 2 tables customer table and products table
	- We now need to consolidate the data of both the tables
	- We can create a new table named orders which would have the data linked to the orders this would have the data of customer and the products that they have ordered.

The process of combining 2 or more tables into one table is called join operations in sql.
